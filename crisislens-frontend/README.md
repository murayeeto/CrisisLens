# CrisisLens Frontend

## Run
npm install
npm run dev        # http://localhost:5173

## Environment
Create `.env.local` if you want to hit a real backend:
VITE_API_BASE=http://localhost:4000

If VITE_API_BASE is unset or any request fails, the app uses rich mock data
so the demo always works.

## Keyboard
⌘K / Ctrl+K   open command palette
Esc            close panel / palette
